## Week 1 Contents

### 1. Design Patterns

#### Singleton Pattern
Implementation of the Singleton Design Pattern to ensure that only one instance of a class exists throughout the application.

#### Factory Method Pattern
Implementation of the Factory Method Design Pattern for creating different types of documents.

### 2. SOLID Principles Notes

Topics Included:

- Single Responsibility Principle (SRP)
- Open Closed Principle (OCP)
- Liskov Substitution Principle (LSP)
- Interface Segregation Principle (ISP)
- Dependency Inversion Principle (DIP)

### 3. Data Structures

#### E-Commerce Platform Search Function

Implementation and comparison of searching techniques for product lookup in an e-commerce platform.

Algorithms Implemented:

- Linear Search
- Binary Search
